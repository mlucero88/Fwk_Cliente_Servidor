Descripción detallada

Clase para manejar los mutex de POSIX threads. Tiene una interfaz muy simple, con los métodos básicos y solo se manejan mutex creados con los atributos por defecto.

Documentación de las funciones miembro

bool PThread::Mutex::bloquear ( )

Método para bloquear el mutex e impedir a otros threads acceder a código salvaguardado por el mismo mutex hasta que sea liberado.

Precondición
El hilo actual no está en posesión del mutex (no lo bloqueó)
Postcondición
Si el mutex se encuentra liberado, el hilo actual se apropia del mismo y lo bloquea. Si ya se encontraba bloqueado, el hilo queda a la espera de que se le pueda asignar el mutex
Devuelve
true si no hay errores al intentar tomar el mutex
false si el hilo ya está en posesión del mutex antes del llamado al método
bool PThread::Mutex::desbloquear ( )

Método para liberar el mutex por parte del hilo en ejecución.

Precondición
El hilo actual debe estar en posesión del mutex
Postcondición
Si el mutex se encuentra bloqueado por el hilo en ejecución, se libera
Devuelve
true si se libera mutex
false si el hilo no estaba en posesión del mutex o el mutex ya estaba liberado
bool PThread::Mutex::intentarBloquear ( )

Método para intentar bloquear el mutex e impedir a otros threads acceder a código salvaguardado por el mismo mutex hasta que sea liberado.

Postcondición
Si el mutex se encuentra liberado, el hilo actual se apropia del mismo y lo bloquea. Si ya se encontraba bloqueado, se retorna false y el hilo actual sigue su ejecución
Devuelve
true si se logra bloquear el mutex
false si el mutex se encontraba ya bloqueado
void PThread::Mutex::signal ( )

Método que envía una señal al los threads que están bloqueados por el método Mutex::wait, avisándoles que se desbloqueen.

Postcondición
Los threads que se encuentran en espera por el método Mutex::wait se desbloquean y continuan su ejecución
void PThread::Mutex::wait ( )

Hace esperar al thread en ejecución hasta que reciba la señal enviada por Mutex::signal.

Postcondición
El thread en ejecución se bloquea hasta recibir la señal de desbloqueo
